#  Chat Bot Project

A simple Python-based chatbot that can interact with users through text input and provide intelligent responses.

---

##  Features

-  Text-based conversation
-  Basic AI responses
-  Predefined question-answer system
-  Fast and lightweight
-  Easy to extend with more commands

---

##  Technologies Used

- Python
- NLP (Basic)
- Standard Python Libraries

---

##  Project Structure

Chat-bot/
│
├── main.py # Main chatbot program
├── responses.py # Response logic (if exists)
├── README.md # Documentation


---

##  Installation

Clone the repository:

```bash
git clone https://github.com/Adan-Ahmed/Chat-bot.git
cd Chat-bot
```

Install dependencies (if any):
```bash
pip install -r requirements.txt
```

 Usage

Run the chatbot:

```bash
python main.py

```

Then start chatting with the bot in the terminal.

 Example Commands
Hello
How are you?
What is your name?
Help
